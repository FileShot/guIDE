Can you explain how the Layout component works?
The Layout component is the main shell of guIDE. It renders the full IDE frame:

• Title bar (34px) — logo, menu, search input, layout toggles
• Activity bar (48px) — left side, icon navigation
• Sidebar (260px) — file explorer, search, git, etc.
• Editor — center, with tab bar and Monaco
• Terminal — bottom of center, resizable
• Chat panel (340px) — this panel, on the right

All dimensions are resizable via mouse drag. Panel visibility uses useState toggles.

How do file icons work?
The FileIcon component in FileExplorer/FileIcon.tsx maps file extensions to SVG icons from public/icons/ (364 icons, file-icon-vectors vivid set).

It has an EXTENSION_ALIASES map for alternate extensions (e.g. tsx → jsx, mjs → js) and falls back to blank.svg if no icon matches.
