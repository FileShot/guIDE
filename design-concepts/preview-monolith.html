Can you explain how the Layout component works?
The Layout component is the main shell of guIDE. It renders the full IDE frame:

• Title bar (28px) — logo, menu, search, layout toggles
• Activity bar (40px) — left, icon navigation
• Sidebar (240px) — explorer, search, git
• Editor — center, tabs + Monaco
• Terminal — bottom, resizable
• Chat (320px) — this panel

All panel dimensions are resizable. Visibility uses useState toggles.

How do file icons work?
The FileIcon component in FileExplorer/FileIcon.tsx maps file extensions to SVGs from public/icons/ (364 icons).

It uses EXTENSION_ALIASES for alternate extensions (tsx → jsx, mjs → js) and falls back to blank.svg.
